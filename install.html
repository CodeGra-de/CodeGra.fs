Installation¶

The CodeGrade Filesystem is an external application that can be installed on Windows, MacOS and Debian/Ubuntu. To install it go to The CodeGrade website and follow the instructions there.

Dependencies¶

The supplied installers automatically install all required dependencies for the CodeGrade Filesystem to work, these are:

  • Python 3

  • Electron

  • FUSE (for MacOS)

  • WinFsp (for Windows)

  • requests

  • fusepy

  • cffi

  • winfspy
